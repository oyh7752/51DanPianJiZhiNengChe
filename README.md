# 51单片机智能车

-----
**基于89C52RC单片机的智能小车**

**模块和传感器**
> * L289N驱动
> * HC-06从机蓝牙模块
> * HC-SR04超声波模块
> * HC-SR04光电反射传感器
> * HC-SR04红外反射传感器
